Strimzi {{site.data.releases.operator[0].version}} documentation (latest stable release)

Strimzi provides operators to deploy and manage Kafka clusters: the Cluster Operator for cluster lifecycle management, the Topic Operator and User Operator to manage topics and users. If you're running the Cluster Operator, you can also use the Access Operator to configure client connections and the Drain Cleaner to assist with pod evictions. Find more details and deployment instructions in the Overview and Deploying and Managing guide.

Strimi's Kafka Bridge offers a RESTful interface for producing and consuming messages. See the Overview and Kafka Bridge documentation for details and deployment instructions.
